##Задание:

 Реализовать игру "Сапер" со следующими возможностями:

 - При выборе клетки без мин указывать число мин в соседних клетках окрестности радиуса 1.
 - При выборе всех клеток без мин вывести надпись "Победа".
 - При попадании на мину вывести надпись "Поражение" и начать игру заново.
