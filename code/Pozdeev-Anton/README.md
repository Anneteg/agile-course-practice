## Задание

Реализовать класс с поддержкой перевода из одного пространства в другое.

Основные операции:

- Перевод:
 - RGB <-> HSV
 - RGB <-> LAB (с использованием дополнительного пространства XYZ)
 - HSV <-> LAB

["RGB -> XYZ"][RGBtoXYZ]

["XYZ -> RGB"][XYZtoRGB]

["XYZ -> LAB"][XYZtoLAB]

["LAB -> XYZ"][LABtoXYZ]
