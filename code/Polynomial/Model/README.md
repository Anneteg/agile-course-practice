## Задание

Реализовать класс полиномов с поддержкой основных операций:

 - Сложение
 - Вычитание
 - Умножение
 - Деление
 - Возведение в целочисленную степень

Все необходимые математические сведения можно найти в статье
["Многочлен"][polynomial] на сайте Wikipedia.
